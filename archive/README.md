# Archive

This directory keeps superseded implementation files that are no longer part
of the active Streamlit route graph but may still be useful for capstone
history or reference.

Files here are not imported by the application. Do not build new features on
them; restore or adapt the relevant code into `app/` only after checking its
imports and current data/model paths.

`deprecated_scripts/` contains superseded command aliases retained so earlier
capstone instructions can be traced without cluttering the active `00-07`
workflow.
